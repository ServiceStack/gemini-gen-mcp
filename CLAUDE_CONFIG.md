# Claude Desktop Configuration

This file is an example configuration for Claude Desktop.

## Installation Instructions

1. Copy the contents of `claude_desktop_config.example.json`
2. Save it as `claude_desktop_config.json` in the appropriate location:
   - **macOS**: `~/Library/Application Support/Claude/`
   - **Windows**: `%APPDATA%\Claude\`
   - **Linux**: `~/.config/claude/`
3. Replace `your-api-key-here` with your actual Gemini API key
4. Restart Claude Desktop

## Getting Your API Key

Get your Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey).
